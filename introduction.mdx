Sift is a course planning platform built specifically for Computer Science students at Texas A&M University. It combines grade distributions, professor ratings, and your academic history to deliver personalized course recommendations.

## What is Sift?

Sift helps CS students answer the critical question: **"What courses should I take next semester?"**

Instead of manually cross-referencing Anex grade distributions, RateMyProfessors ratings, prerequisite chains, and your degree plan, Sift does the heavy lifting for you. Upload your transcript, adjust your preferences, and get ranked recommendations tailored to your academic goals.

## Key Features

### Personalized Recommendations

Sift analyzes multiple data sources to score and rank courses:

- **Grade distributions** from Anex (historical A/B/C/D/F rates by professor)
- **Professor ratings** from RateMyProfessors (quality, difficulty, "would take again" percentage)
- **Prerequisite checking** based on your completed courses
- **Degree plan tracking** to prioritize core requirements, track electives, and capstone courses

### Transcript Parsing

Upload your official TAMU transcript (PDF) and Sift automatically extracts your completed courses using Claude API. No manual data entry required.

<Note>
Transcript parsing uses Claude Haiku for fast, accurate extraction. The PDF text is sent to Claude with strict system instructions to extract only course data.
</Note>

### Customizable Scoring

Adjust how much weight you give to different factors:

- **GPA impact** — prioritize courses with high historical grade averages
- **Professor quality** — favor highly-rated instructors
- **Difficulty** — avoid or embrace challenging courses
- **Degree requirements** — boost core courses vs. electives

Your preferences are saved to your profile and applied to all recommendations.

### Real Course Data

Sift scrapes live data from:

- **Anex** — grade distributions for all CSCE courses since 2015
- **RateMyProfessors** — ratings for 100+ CS department faculty

Data is refreshed via the scraper script (see [Setup](/setup) for details).

  <Step title="Sign up">

## Technology Stack

Sift is built with modern web technologies:

| Layer | Technology |
|-------|------------|
| **Frontend** | Next.js 16 (App Router), TypeScript, Tailwind CSS v4, shadcn/ui, TanStack Query |
| **Backend** | Express.js (TypeScript), Clerk for auth |
| **Database** | Supabase (PostgreSQL) |
| **AI** | Claude API (Anthropic) for transcript parsing |
| **Data Sources** | Anex grade distributions, RateMyProfessors GraphQL API |
